## Propósito

Esta base de conocimiento centraliza la verdad técnica del proyecto:

- **Fuente única de verdad** para arquitectura y patrones de diseño.
- **Guías de on-boarding** para nuevos desarrolladores.
- **Referencias de API** y ejemplos funcionales.

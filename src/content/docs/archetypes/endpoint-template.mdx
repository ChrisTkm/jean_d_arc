# [Dominio] API

Descripción general del grupo de endpoints.

## [Recurso]

### METHOD /api/[ruta]

Descripción de lo que hace el endpoint.

**Query Parameters:**

- `param` (opcional/requerido): Descripción.

**Body (JSON):**

```json
{
  "field": "value"
}
```

**Respuesta Exitosa (200):**

```json
{
"success": true,
"data": { ... }
}
```

**Errores Comunes:**

- `400`: Validación fallida.
- `403`: Permisos insuficientes.

## Reglas de Oro

<CardGrid>
  <Card title="1. Nunca Hardcodear Secretos" icon="warning">
    Todas las credenciales van en variables de entorno.
    `.env` siempre en `.gitignore`.
  </Card>
  <Card title="2. Validar Todo Input" icon="approve-check">
    Usar `express-validator` o Zod en todos los endpoints.
    Nunca confiar en datos del frontend.
  </Card>
  <Card title="3. Queries Parametrizadas" icon="seti:db">
    Siempre usar `$1, $2` en consultas SQL.
    Nunca concatenar strings en queries.
  </Card>
  <Card title="4. Logging Seguro" icon="document">
    Sanitizar logs antes de escribir.
    Nunca loguear passwords, tokens o PII.
  </Card>
</CardGrid>
